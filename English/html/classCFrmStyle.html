Detailed Description

Set style and icon theme form.

Usge:

  • Initialize. Add the following code at the beginning of the program:
    RabbitCommon::CTools::Instance()->Init();
    RabbitCommon::CTools::Init
    void Init(QString szApplicationName=QCoreApplication::applicationName(), QString szApplicationDirPath=QCoreApplication::applicationDirPath(), QString szApplicationInstallRoot=QString(), const QString szLanguage=GetLanguage())
    Initialize the resource, which is called only once at the beginning of the program,...
    Definition RabbitCommonTools.cpp:325
  • Set the style. Use one of the following methods to set up:
    • [Options] Add the menu of set style dialog. Use one of the following methods to set up:
      • [Options] Use RabbitCommon::CTools::AddStyleMenu
      • [Options] Use RabbitCommon::CTools::InsertStyleMenu
    • [Options] In the program you need to open the set style dialog box:
      CFrmStyle* s = new CFrmStyle();
      s->show();
      CFrmStyle
      Set style and icon theme form.
      Definition FrmStyle.h:89

Call QIcon::fromTheme where you need to use the icon

m_pSignalStatus->setIcon(QIcon::fromTheme("newwork-wired"));

If you has your icon theme. then call INSTALL_ICON_THEME() in CMakeLists.txt

# in CMakeLists.txt
# Install QIcon theme
# param SOURCES: Default is ${CMAKE_CURRENT_SOURCE_DIR}/Resource/icons/
# param DESTINATION: Default is ${CMAKE_INSTALL_PREFIX}/data/icons
INSTALL_ICON_THEME()
