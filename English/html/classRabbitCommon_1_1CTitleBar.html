Detailed Description

The cursom title bar for QWidget, QDockWidget etc.

  • The default:
    • title label
    • minimized button
    • maximized button
    • float button
    • close button
  • Add custom buttons with AddWidgets.
  • Usage:
    • If the parent is QDockWidget. use QDockWidget::setTitleBarWidget settings
